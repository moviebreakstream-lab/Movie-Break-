# Movie Break

خدمة احترافية لاستخراج روابط M3u8 من مصادر بث الأفلام والمسلسلات والأنمي، مع استخراج روابط الترجمة وجودة المصادر، وإنتاج قوائم IPTV.

## الميزات:

*   **استخراج M3u8**: استخراج روابط بث الفيديو بصيغة M3u8 من مصادر متعددة.
*   **دعم الترجمات**: استخراج روابط الترجمة بعدة لغات.
*   **تحديد الجودة**: تحديد جودة الفيديو المتاحة (مثل 1080p, 720p).
*   **معالجة متوازية**: استخراج البيانات من مصادر متعددة بشكل متوازٍ لزيادة الكفاءة.
*   **توليد قوائم IPTV**: إنشاء قوائم تشغيل IPTV بصيغة M3u8 جاهزة للاستخدام.
*   **تكامل TMDB**: جلب البيانات الوصفية للأفلام والمسلسلات من The Movie Database.

## الهيكل التقني:

المشروع مبني باستخدام Python ويستخدم المكتبات التالية:

*   `httpx`: لطلبات HTTP غير المتزامنة.
*   `beautifulsoup4`: لتحليل HTML.
*   `cryptography`: لفك التشفير (إذا لزم الأمر).
*   `asyncio` و `aiohttp`: للمعالجة المتوازية.
*   `pybase64`: للتعامل مع Base64.
*   `python-dotenv`: لإدارة المتغيرات البيئية.

## كيفية الاستخدام (قريبًا):

سيتم توفير تعليمات مفصلة حول كيفية إعداد وتشغيل الخدمة، بالإضافة إلى كيفية إضافة مصادر جديدة وتخصيص قوائم IPTV.

## النشر على Render:

تم توفير ملف `render.yaml` لتسهيل نشر الخدمة على منصة Render. يمكنك نشر المشروع كـ "Worker" على Render.

### المتغيرات البيئية:

يجب عليك تعيين المتغيرات البيئية التالية في إعدادات Render الخاصة بمشروعك:

*   `TMDB_API_KEY`: مفتاح API الخاص بك من The Movie Database (TMDB). يمكنك الحصول عليه من [موقع TMDB](https://www.themoviedb.org/settings/api).
